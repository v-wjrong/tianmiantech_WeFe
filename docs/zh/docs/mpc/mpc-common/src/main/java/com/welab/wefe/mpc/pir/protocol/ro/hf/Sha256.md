# 说明

该内容描述了一个名为Sha256的类，实现了HashFunction接口。该类包含一个digest方法，接收字节数组作为输入消息，使用SHA-256算法生成哈希值。方法内部通过MessageDigest类获取SHA-256实例，更新消息并计算摘要。若出现异常，方法返回空字节数组。整个过程展示了SHA-256哈希功能的基本实现。
