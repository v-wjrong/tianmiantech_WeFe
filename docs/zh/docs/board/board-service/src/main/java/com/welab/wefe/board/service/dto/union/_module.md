# 说明

UnionDataSetOutput类是一个包含数据集相关属性的Java类，主要用于存储和管理数据集的基本信息。该类包含成员ID、创建时间、更新时间、维度数量、描述、标签、是否包含Y标志、使用次数、名称、样本数量、唯一ID以及维度列表等字段。每个字段都有对应的getter和setter方法，用于访问和修改属性值。
