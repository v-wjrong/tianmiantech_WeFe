# 说明

该内容定义了一个名为MemberBaseInfo的公开类，用于存储成员基本信息。类中包含五个成员变量：memberId（字符串类型，表示成员ID）、name（字符串类型，表示成员姓名）、hidden（布尔类型，表示是否隐藏）、lostContact（布尔类型，表示是否失联）、freezed（布尔类型，表示是否冻结）。这些变量均为公开可访问，未进行封装。
