# 说明

UserConstant类定义了两个公共静态常量字符串字段。ADMIN_ACCOUNT字段值为"admin"，表示管理员账户名称。DEFAULT_PASSWORD字段值为"wefe123456"，表示默认密码设置。这两个常量用于在系统中统一管理用户相关的固定值。
