
# WeFe

## 概述  
该工程是联邦学习与隐私计算领域的全栈平台，定位为"安全多方协作的分布式AI基础设施"，类似加密版的AI协作工厂。核心解决数据孤岛场景下的联合建模问题，通过多层安全协议（如PSI/PIR/SA）实现"数据可用不可见"，覆盖金融风控、医疗研究等跨机构数据协作场景。  

架构采用微服务分层设计，整合网关层（gRPC通信）、计算层（MPC算法）、管理层（区块链RBAC）和存储层（多数据库适配）。核心资源包括：1) SDK工具链（如PSI客户端/证书管理器）；2) CLI管理控制台；3) 智能合约组件。技术栈聚合Spring生态、国密算法、FISCO BCOS联盟链及机器学习框架（XGBoost/PaddlePaddle），形成从数据对齐→联合训练→安全预测的全流程闭环。  

## 什么是WeFe?  
WeFe是模块化联邦学习操作系统，核心功能包括：1) **安全数据对齐**（基于RSA-PSI协议实现加密样本匹配，类似盲签名流程）；2) **联合建模**（通过分布式XGBoost合并梯度，类似MapReduce的隐私增强版）；3) **区块链治理**（智能合约管理成员准入，类似去中心化工单系统）。  

技术实现依托三层架构：基础设施层（网关/证书服务）、算法层（MPC协议栈）和应用层（RESTful API）。典型工作流如医疗机构A通过网关加密数据→与机构B执行PSI对齐→联合训练逻辑回归模型→结果经区块链审计后返回。模块交互采用"事件驱动+工厂模式"，例如证书签发触发智能合约事件，ClientFactory动态加载验证码服务。  

应用场景模板包括：1) **跨银行反欺诈**（PSI匹配黑名单→安全聚合模型）；2) **医疗研究**（PIR检索病例→联邦学习建模）；3) **政务数据开放**（区块链权限控制→多方计算统计）。关键实现如通过SM2国密算法保障通信安全，BloomFilterUtils加速数据去重，GatewayMetaProto实现gRPC流式分片传输，形成从数据到服务的全链路隐私保护。
